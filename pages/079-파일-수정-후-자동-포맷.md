## 핵심 개념

파일 수정 후 자동 포맷은 `PostToolUse` Hook의 흔한 사용 사례입니다. Claude가 파일을 수정한 뒤 Prettier, gofmt, black 같은 포맷터를 자동으로 실행해 스타일 노이즈를 줄입니다.

단, 포맷은 범위를 좁게 적용해야 합니다. 전체 저장소 포맷을 자동 실행하면 diff가 불필요하게 커질 수 있습니다.

## 설정 예시

```json
{
  "hooks": {
    "PostToolUse": [
      {
        "matcher": "Edit",
        "hooks": [
          {
            "type": "command",
            "command": "python .claude/hooks/format_touched_file.py"
          }
        ]
      }
    ]
  }
}
```

## 포맷 스크립트 예시

```python
import json
import subprocess
import sys
from pathlib import Path

data = json.load(sys.stdin)
path = data.get("tool_input", {}).get("file_path")

if not path:
    sys.exit(0)

p = Path(path)

commands = {
    ".py": ["python", "-m", "black", str(p)],
    ".go": ["gofmt", "-w", str(p)],
    ".ts": ["npx", "prettier", "--write", str(p)],
    ".tsx": ["npx", "prettier", "--write", str(p)],
    ".js": ["npx", "prettier", "--write", str(p)],
}

cmd = commands.get(p.suffix)
if not cmd:
    sys.exit(0)

result = subprocess.run(cmd, capture_output=True, text=True)
if result.returncode != 0:
    print(result.stderr, file=sys.stderr)
    sys.exit(2)

sys.exit(0)
```

## 자동 포맷의 원칙

- 수정된 파일만 포맷한다.
- 포맷터가 프로젝트에 설치되어 있는지 확인한다.
- 실패 시 Claude가 원인을 볼 수 있게 stderr를 전달한다.
- 포맷 변경은 기능 변경 리뷰를 방해하지 않게 관리한다.

## 포맷터 선택 기준

포맷터는 프로젝트가 이미 쓰는 도구를 우선합니다.

| 파일 | 예시 도구 | 확인할 파일 |
| --- | --- | --- |
| JavaScript/TypeScript | Prettier, ESLint | `package.json`, `.prettierrc` |
| Python | black, ruff | `pyproject.toml` |
| Go | gofmt | `go.mod` |
| Kotlin | ktlint | Gradle 설정 |
| Markdown | prettier | 문서 포맷 설정 |

설정 파일이 없으면 자동 실행하지 말고 제안만 하세요. 팀이 쓰지 않는 포맷터를 Hook이 임의로 적용하면 diff가 커질 수 있습니다.

## 실패 시 대응

포맷 실패는 숨기지 말고 Claude가 후속 조치를 할 수 있게 메시지를 남깁니다.

- 포맷터가 설치되지 않았는지 확인한다.
- 설정 파일 오류인지 확인한다.
- 파일 문법 오류 때문에 포맷이 실패했는지 확인한다.
- 전체 저장소 포맷이 필요한 상황인지 별도 작업으로 분리한다.

## 체크리스트

- [ ] 전체 저장소가 아니라 touched file만 포맷한다.
- [ ] 확장자별 포맷터를 명확히 매핑한다.
- [ ] 포맷 실패를 숨기지 않는다.
- [ ] 포맷터 설치 여부를 문서화한다.
- [ ] 큰 포맷 변경은 별도 작업으로 분리한다.
- [ ] 프로젝트에 이미 있는 포맷 설정을 우선한다.
- [ ] 설정이 없으면 자동 실행 대신 제안으로 남긴다.
